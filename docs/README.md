## < 체크 리스트 >

- 자바 버전 17로 맞추자. ( ✅ )
- build.gradle를 바꾸지 말자. ( ✅ )

- 시작점은 main에서 시작하도록 하자.
- "JAVA 코드 컨벤션" 을 준수하자.
- 주어진 라이브러리를 사용하자.
- 요구사항에 명시된 출력형식을 지키자.
- 테스트를 돌려서 성공하자. ( ./gradlew.bat clean test )
- 구현이 끝나면 미션 제출방법을 확인 하자.
